# NEU WakeUp课表导出（Android）

针对金智教育教务系统的课程表导出至第三方课程表的 Android 工具。 
（以东北大学本科教务管理系统(2026更新)为例）
用于在手机端登录教务系统并导出 WakeUp 可导入的 CSV 课表文件。

本项目受 [CreamPig233/neu_wisedu2wakeup](https://github.com/CreamPig233/neu_wisedu2wakeup) 项目的启发，将其 Python 逻辑移植为安卓客户端，适用于不方便/不熟悉使用电脑的Android手机用户。

> [!CAUTION]
> 截至目前(2026-2-15)，新教务系统仍在持续更新，本工具仅提供辅助作用，如果课程表有改动，请**时刻**以教务系统中显示的为准！

## 功能
- 自动检测网络模式：
- 校园网直连
- 外网自动切换 WebVPN
- 原生解析课表接口并导出 CSV
- 导出时默认定位到 `Documents` 目录（系统文件选择器）

## 使用方法
1. 打开 App，等待“网络检测完成”。

2. 在下方官方页面完成教务系统登录。

3. 点击“检测登录状态”，确认已获取用户与学期信息。

4. 学期代码可自动填充，也可手动修改（示例：`2025-2026-2`）。

5. 点击“获取课表”。

6. 点击“导出 CSV”，保存到 `Documents`。

7. 在 WakeUp 课程表中选择“从 CSV 导入”并选中该文件。

   注：如果自动使用了文档编辑器类应用或其他非WakeUp课程表应用打开，说明您曾指定过csv文件的默认打开应用，此时需要按照WakeUp官方教程从手机中手动导入csv文件。[如何导入？](https://wakeup.fun/doc/import_from_csv.html)


## 隐私与安全
- 不接入第三方后端，不上传账号或课表到开发者服务器
- 数据请求直接由你的手机发往学校系统（或 WebVPN）

## 开发环境
- Android Studio（Koala 或更新）
- JDK 11
- Android SDK 36
- Min SDK 24
